Port Scanner

Usage

  1. Enter the target host in the 'Host' input field.
  2. Enter the target ports in the 'Ports' input field, separated by commas (e.g., 80,443,22).
  3. Click the 'Scan' button to start the port scan.

Help

  1. This application performs a port scan on a specified host.
  2. You need to provide the host and a list of ports to scan.
  3. The scan will check if the specified ports are open on the target host.
  4. Syntax for Enter Host : example.com OR IPV4 (0.0.0.0) .
  5. Ports range start from 1 and ends on 65535 you can scan one or many ports between this range.
  6. Syntax for Enter Ports : 80 OR 80,443,53,21,....
